Traffic Sign Detection & Recognition

This application detected traffic signs around 50 m, Due to traffic the road accidents was increasing in the world.
In this I used 43 different signs.The project uses various python libraries including stream lit , Cv2,
numpy, tensorflow,etc. Firstly, it can accurately detect and recognize traffic signs,
which is essential for ensuring road safety. Secondly, it can operate in real-time,
providing drivers with instant information about traffic signs.
Finally, the system is scalable and adaptable,meaning
that it can be applied to different
types of roads and environments.


python libraries used in project as follows:

1] Streamlit: Streamlit is a primary framework for developing
the user interfaces Streamlit is a popular Python framework
that simplifies the process of building web applications
2] CV2: CV2 is used for image pre-processing and post-processing
operation including image resizing, conversion to grayscale,
and drawing bounding boxes around detected traffic signs.
3] TenserFlow: TensorFlow is an open-source software library
used in many applications like speech recognition, natural
language processing. TensorFlow is used as the backend allowing
for efficient training and evaluation of the model.
4] NumPy: NumPy is a Python library used for numerical
computations in scientific computing.It provides a
multidimensional array object,as well as tools for
working with these arrays.NumPy is used for various tasks,
including image processing, data manipulation,etc.
